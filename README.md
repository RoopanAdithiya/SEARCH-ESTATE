# 🏙 SEARCH ESTATE

An AI-powered rental intelligence platform for Chennai.

## 🚀 Features

- 🧠 Rent prediction using ML (XGBoost)
- 📊 SHAP explainability (WHY this price)
- 🏠 Property listing system
- 📉 Overpricing detection
- 💸 Rent stress analysis

## 🧱 Tech Stack

Frontend:
- React (Vite)
- Tailwind CSS
- Recharts

Backend:
- FastAPI
- SQLite
- XGBoost
- SHAP

## ⚙️ How to Run

### Backend
cd backend  
pip install -r requirements.txt  
uvicorn main:app --reload  

### Frontend
cd frontend  
npm install  
npm run dev

An explainable AI-powered rental marketplace.
